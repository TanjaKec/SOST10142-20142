# SOST 10141 / 20142 – Applied Statistics for Social Scientists

This repository contains teaching materials for the course  
**SOST 10141 / 20142 – Applied Statistics for Social Scientists**.

The materials will be added **week by week throughout the semester** and include lecture notes, practical exercises, datasets, and supporting resources used in class.

The aim of this repository is to support **reproducible statistical analysis** and allow students to revisit the examples demonstrated during lectures and practical sessions.

---

## Course Topics

The course introduces statistical methods commonly used in the social sciences, including:

- exploratory data analysis  
- visualising relationships between variables  
- correlation analysis  
- simple linear regression  
- multiple regression  
- interpretation of statistical models  

All examples are implemented using **R and Quarto**.

## Intro to Regression Materials

The folder **Intro2Regression** contains materials for the regression section of the course, including:

- lecture notes introducing regression concepts  
- practical exercises analysing real datasets  
- example code used during demonstrations  

These materials introduce key ideas such as:

- scatterplots and visualising relationships  
- the regression line  
- residuals  
- explained and unexplained variation  
- the coefficient of determination ($R^2$)  
- hypothesis testing using the F-test  
- interpreting regression output in R  

---

## Reproducibility

All examples in this repository are designed to be **fully reproducible**.

Students should be able to reproduce the analysis by:

1. downloading or cloning the repository  
2. opening the files in **RStudio**  
3. running the provided code  

Required packages are typically loaded within the documents.

---

## How Students Should Use This Repository

Students are encouraged to:

- download or clone the repository  
- run the code examples in RStudio  
- modify the code to explore additional relationships in the datasets  
- use the materials when revising course concepts
